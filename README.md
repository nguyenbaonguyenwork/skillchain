# SkillChain

Decentralized cross-border skill identity platform built on Stellar.

---

## Problem

Traditional certificates and CVs:
- can be faked
- difficult to verify
- fragmented across platforms

---

## Solution

SkillChain provides:
- on-chain credentials
- skill passport
- transparent verification
- scholarship automation

---

## Features

- Stellar Wallet Integration
- Credential Issuance
- Credential Verification
- Skill Passport
- Reward Automation

---

## Tech Stack

- Stellar
- Soroban
- Next.js
- TailwindCSS
- IPFS

---

## Vision

Building a borderless and verifiable skill identity system.
